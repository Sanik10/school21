# 🚀 Ключевые Java-проекты

## 🔐 [APJ1-Backend](./APJ1-Backend)

REST API на Spring Boot с PostgreSQL и JWT-авторизацией.  

Подпроекты:
- [T04 — База данных + Basic Auth](./APJ1-Backend/APJ_Jv_T04B)
- [T05 — JWT + история игр + лидерборд](./APJ1-Backend/APJ_Jv_T05B)

### Реализовано:

- Подключение PostgreSQL
- Spring Data (CrudRepository)
- JPA-аннотации (`@Entity`)
- Basic Auth → переход на JWT
- Access + Refresh токены
- Claims (UUID, роли)
- Role-based доступ
- Custom AuthFilter (GenericFilterBean)
- SecurityFilterChain конфигурация
- SecurityContextHolder
- Многослойная архитектура (web / domain / datasource)
- REST API для:
  - регистрации и авторизации
  - создания и обновления игры
  - истории игр
  - таблицы лидеров
- SQL-запрос для вычисления win-rate пользователей

### Демонстрирует:

- Работа с реляционной БД
- Stateless-аутентификация
- Настройка Spring Security
- Разделение ответственности
- Реализация бизнес-логики поверх REST API
